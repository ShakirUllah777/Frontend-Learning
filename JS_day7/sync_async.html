Sync Vs Async Function in Js.

Sync Function

Function that run the code one by one in the seq flow

Async Function

Function that can be run Parallel. while one code in giving the output and second is run and other is waiting for the output.

Important Note!

While one Code is waitng other can run easily and the waing code will give output after timeinterval.

awit - in Async function

if we want that one line after before its line must run then we used the awit in async function

mostly used in FetchApi and calling data from the DataBase.

When one code output depent on the other one.
